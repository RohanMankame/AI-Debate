# AI Debate Frontend (React + Vite)

A modern, responsive UI designed to simulate a "stage" environment for AI debates.

## Features
- **Dynamic Setup**: Configure topics, round counts, and team personas.
- **Real-time History**: Arguments stream in as they are generated by the backend.
- **Glassmorphic UI**: Clean, dark-mode design with readable typography.

## Development
```bash
npm install # Install dependencies
npm run dev # Start dev server
npm run build # Build for production
```

## Connection
The frontend expects the backend to be running at `http://localhost:8000`. You can modify the `API_URL` in `src/components/SetupDebate.jsx` and `src/components/DebateStage.jsx` if your backend port changes.
